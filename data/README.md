# Data Directory

Place the raw workbook here before running the pipeline from scratch:

`data/raw/online_retail_II.xlsx`

The project does not commit the original Excel file.

You can also override the path by:

- passing `--raw-excel /path/to/online_retail_II.xlsx`
- setting `FORTUNETELLERS_RAW_EXCEL=/path/to/online_retail_II.xlsx`
